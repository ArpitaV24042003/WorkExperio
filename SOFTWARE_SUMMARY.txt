4. MongoDB 7 (Optional)
5. Docker & Docker Compose (Optional - for containerized deployment)

---------------------
- FastAPI 0.114.2
- Uvicorn 0.30.6
- SQLAlchemy 2.0.36

DEPENDENCY FILES IN PROJECT:
----------------------------
- backend/requirements.txt (Python dependencies)
- ai_service/requirements.txt (Python dependencies)
- frontend/package.json (Node.js dependencies)
- frontend/package-lock.json (Locked versions)
- docker-compose.yml (Container configuration)

INSTALLATION COMMANDS:
----------------------
Backend:
  cd backend
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt

Frontend:
  cd frontend
  npm install

AI Service:
  cd ai_service
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt
